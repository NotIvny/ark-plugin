## OCR 圣遗物识别

本插件提供了基于 OCR 的圣遗物识别功能，支持通过截图自动识别圣遗物信息并应用到角色面板中，支持原神和星铁。


### 使用方式

在使用 `#xx面板变换` 相关命令时，附带圣遗物截图即可触发 OCR 识别。插件会将图片发送至 API 进行识别，并自动将识别结果应用到对应的圣遗物位置。

示例：

`#雷神面板换花` + 附带圣遗物截图（可一次性带多图或引用回复）


### 注意事项

- 对于裁剪剩余圣遗物属性的图片分辨率不限（能看清就行）；对于整张未裁剪的图片，Ark OCR API要求每张图片的分辨率不低于360p（过低会显著降低识别成功率），每张图片的处理时间约为3s，后续可能会开放高速API（2x-4x速度）
